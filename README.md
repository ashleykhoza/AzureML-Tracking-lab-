# AzureML-Tracking-lab-
This repository contains a hands-on lab for tracking model training using MLflow within an Azure Machine Learning workspace. I provisioned an Azure ML workspace, set up compute resources, and used Automated Machine Learning. I logged parameters, metrics, and artifacts, giving me a clear, comprehensive view of model performance. 
As a aspiring data scientist, I leveraged the Azure ML platform to provision a central workspace, configure compute resources, and employ Automated Machine Learning for classification tasks. By integrating MLflow into the notebook environment, I systematically logged hyperparameters, performance metrics, and artifacts, creating a transparent record of each experimental run. The workflow began by provisioning the workspace via Azure CLI, followed by cloning lab materials and installing essential Python libraries such as the Azure ML SDK, MLflow, and scikit-learn. Once the setup was complete, I ran a notebook that authenticated with the workspace, trained models, and logged each run’s outcomes. This approach provided a structured framework for comparing different models, ensuring reproducibility, and effectively managing the iterative process of data-driven model development.
